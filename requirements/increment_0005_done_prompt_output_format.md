# Increment 0005: Prompt Output Format

**Requirement ID:** R5  
**Status:** TODO

## Description

Generated prompts must:
- Be actionable and specific
- Include context about current state
- Prioritize based on fitness scores
- Provide clear acceptance criteria

## Related Principles

- [G1 — Context-Aware](../principles/G1.md): Include state summary, metrics, trends
- [G2 — Actionable](../principles/G2.md): Immediately actionable with specific targets
- [G3 — Prioritized](../principles/G3.md): Ranked by impact, effort, dependencies
- [G4 — Measurable](../principles/G4.md): Each prompt has testable acceptance criteria
- [M3 — Transparency](../principles/M3.md): Explain why the prompt was generated

## Acceptance Criteria

- [ ] Prompts contain actionable, specific instructions
- [ ] Prompts include current state context
- [ ] Prompts are ordered by priority (fitness-based)
- [ ] Each prompt has clear acceptance criteria
